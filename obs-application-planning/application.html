1 OBS (Open Broadcaster Software)

Open Broadcaster Software 是由 OBS Project 開發的自由開源跨平台串流媒體和錄影程式。

1.1 應用

1.1.1 錄製應用程式畫面

  • 可以將桌面上的應用程式的畫面，如 PowerPoint 等簡報。
  • 將應用程式操作的畫面錄製。

1.1.2 整合錄影設備及影片場景

  • 將攝影機中的影像整合到桌面的畫面中。
  • 配合已經準備好的影片，並錄製於畫面中。

1.1.3 將上述的整合畫面使用於視訊會議

  • 配合虛擬相機給視訊會議的鏡頭。

1.1.4 網路直播

  • 使用 youtube 等網路平台作為直播工具。
